# 注意：前端调用后端接口后，先检查http响应码是不是200,如果不是的话返回数据的格式都是如下所示，只有一个message！

```
{
  "message": "string"
}
```

# 新书上传(zhj finished)

## /admin/uploadNewBook 方法：post form-data

### 参数：

```
author:string
bookcoverimage:文件
description:string
isbn:string
name:string
publicationDate:string Pattern=("yyyy-MM-dd")
```

### 样例输出：

#### 200,400

```
{
  "message": "string"
}
```

#### 403

说明用户权限不足，只有管理员才能上传书本
